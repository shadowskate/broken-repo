# Assignment 1 : Data management of the sensor logger

## Description : 
The C++ program consists of file handling, directory creation,
logging and random data generation.It simulates the behaviour
of a data logging application by generating random 'sensor' readings
and organizing them into files and directories.

## Output Directory : 
The src/output/ directory is the designated location for all the 
programs result. All the reading generated/recorded during the program's
execution are stored in this directory.
